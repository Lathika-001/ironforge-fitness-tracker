# IronForge Fitness Tracker

IronForge is a modern web-based fitness tracking application that helps users manage workouts, track progress, and stay consistent with their fitness goals.

## Features
- User authentication (Sign Up / Login)
- Workout tracking
- Meal logging system
- Before & After progress photo uploads
- Track fitness progress over time
- Clean and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Tailwind CSS
- Vite
- Supabase (Authentication & Database)

## Installation

Clone the repository:

git clone https://github.com/Lathika-001/ironforge-fitness-tracker.git

Install dependencies:

npm install

Run the project:

npm run dev
